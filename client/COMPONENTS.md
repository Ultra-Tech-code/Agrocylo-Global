# Components

## Shared Form Components
- `FormInput`: text/number/date fields with inline error rendering.
- `FormSelect`: select fields with standardized styling and errors.
- `FormTextarea`: multi-line input with validation feedback.
- `FormErrorSummary`: top-level form error list for quick recovery.

## Error UI Components
- `ErrorBoundary`: global recovery UI with category-based messaging.
- `ErrorDisplay`: inline actionable error card with docs link.

## Provider Stack
- `GlobalProvider` composes theme, query, transaction feedback, wallet/profile/cart.
