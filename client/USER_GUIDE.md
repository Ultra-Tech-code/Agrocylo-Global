# User Guide

## Account and Wallet
- Connect wallet before creating escrow orders or barter offers.
- If wallet errors occur, unlock the wallet and retry.

## Form Validation
- Forms validate in real time.
- Field-level errors appear below inputs.
- Form-level summary lists all blocking issues.

## Error Recovery
- Error screens include:
  - What happened
  - What to do next
  - Troubleshooting documentation links
