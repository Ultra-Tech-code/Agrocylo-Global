# Quick Start

## Install
```bash
npm ci
```

## Run
```bash
npm run dev
```

## Validate
```bash
npm run lint
npm test -- --run
npm run build
```

## Key Paths
- Error handling: `src/lib/errorHandler.ts`
- Logging: `src/lib/logger.ts`
- Validation schemas: `src/lib/validation.ts`
- Form components: `src/components/forms/*`
- Contexts: `src/context/*`
